How big should my recovery partition be?Câu hỏi

Answer jpg. Creating a basic recovery drive requires a USB drive that is at least 512MB in size. For a recovery drive that includes Windows system files, you’ll need a larger USB drive; for a 64-bit copy of Windows 10, the drive should be at least 16GB in size.
